# Test 03: 无图片文章 → Image Sufficiency = "none"

## 场景

纯文字中文内容，无任何图片。测试 image sufficiency 判定为 "none" 后的推荐流程和用户交互。

## Prompt

```
把以下内容做成演示文稿：

# 为什么远程工作正在重塑城市格局

2020年以来，远程工作从临时措施变成了永久趋势。这一变化正在深刻改变城市的经济地理。

## 人口迁移的新模式

旧金山在2020-2023年间流失了约7.5%的人口。与此同时，博伊西、奥斯汀、迈阿密等"第二梯队城市"经历了前所未有的人口涌入。

这不是简单的"逃离大城市"。数据显示，离开的主要是科技工作者——收入最高、消费能力最强的群体。他们带走的不仅是税收，还有整个消费生态。

## 三个正在发生的结构性变化

**第一，商业地产的价值重估。** 曼哈顿的办公楼空置率达到22.7%，创历史新高。一些写字楼的估值已经下跌40-60%。这不是周期性波动，而是需求的永久性萎缩。

**第二，城市中心的功能转型。** 当办公不再是CBD的核心功能，城市中心需要重新定义自己。有些城市开始将写字楼改造为住宅，有些在发展"15分钟城市"概念。

**第三，区域经济的重新平衡。** 远程工作打破了"好工作只在大城市"的定律。一个在博伊西的工程师可以拿到旧金山80%的工资，但生活成本只有40%。这种套利正在重新分配财富。

## 被低估的风险

但这个趋势也有暗面。当所有人都涌向同一批"宜居城市"，这些城市的房价也在快速上涨。博伊西的房价在三年间翻了一番。"逃离高房价"最终可能只是把高房价带到了新地方。

## 展望

远程工作不会消失，但它的影响远未到达稳态。未来五年，我们可能会看到一种新的均衡：大城市缩小但更专注于面对面协作的高价值活动，中小城市承接更多日常工作。

赢家不是最大的城市，也不是最小的城市，而是最善于适应这种新模式的城市。
```

## 预期行为

1. Phase 1 Step 3：image audit 发现零图片
2. Image sufficiency = "none"
3. 触发 Phase 1.5：生成推荐表格（可能建议城市照片、地图、对比图等）
4. 用 AskUserQuestion 问用户是否提供图片
5. 用户选"不需要图片"后，用纯排版方案继续

## 测试交互选择

- 图片推荐：选 B（"不需要图片，直接生成"）
- 风格选择：选 A
- Slide 计划确认：说"可以"

## 重点检查

- Image sufficiency 正确判定为 "none"
- 推荐表格有合理的建议（如旧金山/博伊西的城市照片、商业地产照片）
- 推荐不包括 thesis/stat slide 的图片（这些不需要图片）
- 用户选"不需要"后，没有任何 placeholder 或 stock photo URL 出现
- 数据点有 stat slide（7.5%、22.7%、40-60%）
- "三个结构性变化"有合理的 slide 拆分（不是一个 bullet list）
